# Installation

Sehr geehrte Teilnehmende,

um einen reibungslosen Start zu gewährleisten benötigen wir einige Werkzeuge.
Diese Werkzeuge sind "unverzichtbar" innerhalb der TYPO3 Extension Entwicklung.

Du kannst dir aussuchen ob du die Werkzeuge manuell installierst und entsprechend eigenständig darum kümmerst, dass diese aktuell sind oder du kannst einen meiner Container verwenden.

Im Fall einer "klassischen Installation" befolge bitte die Anweisungen aus der Datei Classic.md

Im Fall von containerisierter Umgebung, auch bekannt als Docker, DDEV oder Podman, befolge bitte die Anweisung aus Docker-based.md
